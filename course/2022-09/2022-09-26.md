# Services and IPtables
1. Create a basic Pod:

    ```
    apiVersion: v1
    kind: Pod
    metadata:
      name: httpd-po-1
      labels:
        app: httpd
    spec:
      containers:
        - name: httpd-container
          image: 'image-registry.openshift-image-registry.svc:5000/openshift/httpd:latest'
          ports:
            - containerPort: 8080
          volumeMounts:
            - mountPath: /var/www/html/
              name: httpd-volume
      initContainers:
        - name: httpd-init
          args:
            - 'touch index.html ; sed -i s/^$/httpd-po-1/ index.html'
          command:
            - sh
            - -c
          image: busybox
          volumeMounts:
            - mountPath: /data/
              name: httpd-volume
          workingDir: /data/
      volumes:
        - name: httpd-volume
    ```
